Technologies used:

Bid Mart

Bid Mart is an online auction platform that brings buyers and sellers together. It allows sellers to list items they want to sell and buyers to bid on items and manage their bids. The platform is built using Node.js, Express.js, MongoDB, HTML, CSS and Bootstrap. The app also takes advantage of the sendgrid api to send email notifications as well as the mapbox api for map functionalities. Features include user authentication, bidding history, user profile management, listings management, auction updates and notifications via email, forgot/reset password. Users can also add items they want to keep an eye on by adding to a watchlist. Asynchronous errors are properly handled.

Typing Tutor Game

This project was part of my coursework in my second year of study. It involved extending a multithreaded educational “Typing Tutor” game coded in Java, with the goal of increasing understanding of thread synchronization and thread safety. The game challenges the player to type in the falling words before they hit the bottom of the screen. As part of the project, two new features were added to the existing game without breaking the existing code: ensuring that the lowest word disappears first when duplicate words appear on the screen, and a green word (Hungry Word) that moves horizontally across the middle of the screen and causes any words it bumps into to disappear. Though this project, I gained experience in working with multithreaded programs and implementing thread synchronization.

Alphabet app for kids

As part of a Mobile Device Development course, I developed an Alphabet Book Android App that allows users to view images corresponding to each letter of the alphabet. The app features two main views: an overview page with a button for each letter of the alphabet and a letter page that displays the selected image. On the letter pages, there are buttons for navigating to the first, previous, next, last letter pages, as well as the overview page that shows all the letters of the alphabet. The app is targeted towards parents and educators of young children. tested it with a Google Pixel 4A emulator running Android 10 (API Level 29)
